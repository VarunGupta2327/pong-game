🏓 Pong Game (HTML, CSS, JavaScript)

A simple Pong Game built using HTML, CSS, and JavaScript.
This project recreates the classic arcade game where two players control paddles and try to bounce the ball past the opponent.

🎮 Features

Classic Pong gameplay

Score tracking

Responsive canvas design

🛠️ Technologies Used

HTML5 – Structure of the game

Canvas API – Rendering the game
